# YouTube Downloader

A web-based application to download YouTube videos and playlists, built with Flask, HTML, and CSS.

## Features

- Download individual YouTube videos
- Download entire YouTube playlists
- Select video quality (highest or lowest)

## Technologies Used

- Python
- Flask
- Pytube
- HTML
- CSS

## Requirements

- Python 3.6+
- Flask
- Pytube
